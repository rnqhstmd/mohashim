## [1.0.0] - 2026-05-12

### 추가

#### 집중도 측정
- 키보드·마우스 활동 점수(최대 80점)와 마이크 주변 소음 환경 점수(최대 20점)를 1초마다 복합 분석하는 하이브리드 집중도 측정 (0~100점)
- 180초 이내 자리비움은 집중 중으로 처리해 활동 만점 유지
- 마이크 dB EMA 기반 환경 점수 산출 (65dBSPL 이하 만점, 80dBSPL 초과 시 0점)
- 실시간 소음 측정기 — 마이크 dB 수준 시각화

#### 포모도로 타이머
- 집중(기본 25분) → 휴식(기본 5분) → 세션 완료 자동 전환 사이클
- 집중/휴식 시간 자유 설정
- 세션 완료 시 OS 알림 발송
- 세션 중단 시 잔디 미기록 처리

#### 잔디 그래프
- 최근 28일 집중 세션 GitHub 스타일 히트맵 시각화
- 일별 세션 수·평균 집중도에 따른 색상 농도 표현
- 날짜별 상세 집중 기록 확인

#### 감자 캐릭터 '모하'
- 집중도에 따른 5단계 표정 반응 (집중 / 평온 / 산만 / 숨김 / 스트레스)
- 표정 단계별 맞춤 멘트 출력
- 메뉴바(macOS) / 트레이(Windows) 아이콘 실시간 연동

#### 새싹 경제 시스템
- 세션 완료 점수 구간에 따라 새싹 1 / 3 / 5개 획득
- 매일 첫 접속 시 출석 보상 새싹 1개 지급
- 상점 — 스킨, 표정 등 9종 캐릭터 아이템 구매
- 캐릭터 레이어 시스템 — 구매 아이템 실시간 적용 및 장착/해제

#### 편지함
- 세션 완료 편지 — 집중 점수 구간별 맞춤 칭찬 및 응원 메시지
- 출석 보상 편지 — 새싹 보상과 함께 도착하는 출석 환영 메시지
- 영수증 편지 — 상점 아이템 구매 시 발급되는 구매 영수증

#### 할 일 관리
- 태그 기반 할 일 분류 및 위치 태그 추가
- EMA(지수이동평균) 작업 점수로 꾸준한 완료 패턴 추적

#### 월간 인사이트
- 태그별 집중 패턴 분석
- 월간 집중 리포트 제공

#### 잔디 자랑하기
- 한 달 잔디 그래프를 1080×1080 PNG로 합성
- 베스트 집중일·최다 할 일 완료일 통계 포함
- 합성 이미지 클립보드 복사

#### 시스템 및 기타
- 부팅 시 자동 시작 설정
- 설정에서 최신 버전 확인 및 앱 업데이트
- JSONL 형식 로컬 분석 로그 (app_start, session_complete, sprout_earned 등 이벤트 기록)
- 완전 로컬 저장 — 모든 데이터를 사용자 기기에만 보관, 외부 서버 전송 없음
- 마이크 음성 미저장 — dB 수치만 실시간 분석
