# todo 용어 사전

| 용어 | 설명 |
|------|------|
| 투두 (Todo) | 할 일 항목 1건. 텍스트 + 작업 태그(선택) + 위치 태그(선택) + 완료 여부 |
| 작업 태그 | 할 일의 카테고리 (예: 개발/공부/디자인/독서/글쓰기/잡무). 이모지 + 색상 + 라벨 |
| 위치 태그 | 할 일의 작업 장소 (예: 집/카페/사무실/도서관). 이모지 + 색상 + 라벨 |
| 자동 정렬 | 미완료 항목 = 목록 상단, 완료 = 취소선 + 최하단 이동, 롤백(완료→미완료) = 즉시 최상단 복귀 |
| Active Todo | 사용자가 다음 뽀모도로에서 할 항목으로 표시한 미완료 1개. 노란 하이라이트 |
| 태그 편집기 | 추가/삭제/이름·이모지·색상 변경 가능한 화면. 작업/위치 각각 별도 |
| 기본 작업 태그 | 개발 / 공부 / 디자인 / 독서 / 글쓰기 / 잡무 (6종) |
| 기본 위치 태그 | 집 / 카페 / 사무실 / 도서관 (4종) |
| 작업 태그 maxItems=5 | 작업 태그는 최대 5개까지만 등록 가능. 위치 태그는 제한 없음 |
| MetaChip | 태그 칩 컴포넌트. 이모지 + 라벨, active 시 배경 컬러 채움 + 흰 글자 |
| FlatTag | 투두 항목 행에 보이는 미니 태그. 컬러 닷 + 라벨, 보더 없음 |
| EmptyChip | "선택 안 함" placeholder 칩. dashed 보더 + ＋ 아이콘 |
| TagListEditor | 작업 태그·위치 태그 편집 공통 컴포넌트. 각 항목마다 이모지 picker(18종) + 색상 picker(10종) + 인라인 이름 수정 + 삭제 |
| Active 시각 표현 | 노란 그라디언트 배경 (`#fff8e0` → `#fff2c4`) + `ACCENT_DARK` 왼쪽 막대(4px×24px) + 그림자 |
| 이모지 picker 18종 | 💻 📚 ✏️ 📖 ✍️ 📋 🎨 🧠 🏃 🍳 🎵 🏠 ☕ 🏢 📍 🌳 🎯 💡 |
| 색상 palette 10종 | `#7aa3e6 #9d7ad9 #f4a261 #5fa97a #d68a6a #a8b3cc #7dc89a #5a8dd8 #c46455 #f4d160` |
