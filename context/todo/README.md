# todo (투두 + 태그)

뽀모도로와 **완전히 독립적**으로 작동하는 할 일 관리. 작업 태그·위치 태그 시스템과 자동 정렬.

## 배경

집중 측정과 별개로, 사용자가 오늘 무엇을 할지 정리하는 공간이 필요. 기존 투두 앱과 차이는 **작업 태그(개발/공부/디자인…) + 위치 태그(집/카페/도서관…)** 를 붙여 통계 분류와 연결할 수 있다는 점.

## 안 하면 어떻게 되는가

투두 없이 뽀모도로만 있으면 "뭘 할지 모르고 타이머만 돌아가는" 흐름이 됨. 사용자 진입점이 약해진다.

## 사용자와 규모

본인 1인.

## 성공 기준

- ✅ 자동 정렬: 미완료 상단, 완료 즉시 취소선+최하단, 롤백 즉시 최상단 복귀
- ✅ 태그 추가/삭제/이름·이모지·색상 편집
- ✅ 할 일 등록 시 작업 태그 1개, 위치 태그 1개 선택 가능
