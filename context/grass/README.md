# grass (잔디 + 공유 카드)

28일 윈도우 일자별 세션 기록을 GitHub 스타일 격자로 시각화 + 정방형 공유 카드(`<ShareCard>`) 합성.

## 배경

뽀모도로의 결과물을 한눈에 보여주는 잔디 + 자랑할 수 있는 1080×1080 정방형 이미지. 세션 단위로 적재되므로 도중 취소·슬립 discard는 미반영.

## 안 하면 어떻게 되는가

행동의 누적 가시성이 사라지면 동기 유지 어려움. 잔디 자랑하기는 외부 공유로 자연 유입 효과까지 노린 기능.

## 사용자와 규모

본인 1인.

## 성공 기준

- ✅ 28일 윈도우 잔디 레벨 0~4 정확히 매핑
- ✅ 세션이 완전히 끝났을 때만 평균 점수 적재 (도중 취소 미적재)
- ✅ 공유 카드는 화면 캡처가 아닌 **전용 SVG 템플릿** 합성
- ✅ 1초 이내 PNG 합성 → 클립보드 복사
