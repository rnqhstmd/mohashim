# character (모하 캐릭터 + 멘트)

감자 캐릭터 '모하' 5단계 표정과 츤데레/플러팅 멘트 8버킷. 점수와 모드에 따라 표정·멘트가 자동 분기.

## 배경

측정 신뢰성과 함께 모하심의 또다른 핵심 가치. 혼자 일하는 사용자에게 "누가 같이 있어주는 느낌"을 주고, 점수 변화를 캐릭터 표정으로 즉각 피드백한다.

## 안 하면 어떻게 되는가

캐릭터·멘트가 빠지면 단순 측정 도구로 전락. 잔디 자랑·재방문 동기 모두 약화.

## 사용자와 규모

본인 1인.

## 성공 기준

- ✅ 5단계 SVG가 점수 구간(0~100)에 정확히 매핑
- ✅ 새싹(sprout)이 상태별로 활기 → 시듦 변화
- ✅ 8개 멘트 버킷이 phase·dB·종료 상태에 따라 정확히 선택
- ✅ Idle 동안 7개 라벨이 8초 주기로 회전 (`character` 컴포넌트와 `timer` chip 동기)
