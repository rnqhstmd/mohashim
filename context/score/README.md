# score (점수 엔진)

작업 활동 80점 + 소음 환경 20점 = 0~100점 하이브리드 집중도 측정. **Rust 백그라운드 상시 실행** 엔진.

## 배경

행동 기반의 신뢰성 있는 점수가 모하심의 핵심 차별점. 단순 체크 토글로는 잔디가 자기만족 수치에 그친다. 키보드/마우스 발생 여부(내용 비수집) + 마이크 dB(EMA 필터)로 측정한다.

## 안 하면 어떻게 되는가

점수 신뢰성이 무너지면 잔디·통계의 의미가 사라지고 단순 뽀모도로 클론과 구분 불가.

## 사용자와 규모

본인 1인 (개인 프로젝트). MVP 검증은 본인 사용 단계.

## 성공 기준

- ✅ 작업 점수 80점 만점, grace 3분 정확
- ✅ 소음 점수 dB 구간(0~65=20, 66~80=비례, 80+=0) 정확
- ✅ 5단계 매핑이 트레이/팝업에서 일관
- ✅ 입력 내용 비수집을 코드 레벨에서 강제
