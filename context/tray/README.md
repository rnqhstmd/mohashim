# tray (트레이/메뉴바)

macOS 메뉴바 / Windows 시스템 트레이에 5단계 모하 아이콘을 상주시키는 도메인. 마스터 SVG 1세트 → OS별 빌드 변환.

## 배경

데스크탑 앱의 진입점이자 항상 보이는 시각 요소. 점수에 따라 표정이 바뀌어 사용자가 작업 중에도 글랜서블한 피드백을 받는다.

## 안 하면 어떻게 되는가

트레이가 정적이거나 OS별로 부자연스러우면 데스크탑 네이티브 앱의 가치가 약해진다.

## 사용자와 규모

본인 1인.

## 성공 기준

- ✅ macOS = template 이미지 (라이트/다크 자동 반전)
- ✅ Windows = 컬러 ICO (멀티 해상도)
- ✅ Score Engine 5단계 변경 시 즉시 아이콘 갱신
- ✅ 뽀모도로 진행 중에만 시간 텍스트("18:22") 동봉
- ✅ Idle 모드는 `calm` 고정 + dB>80 시 `stressed` (DEC-6)
