# lifecycle (앱 라이프사이클 + Storage + 폰트)

권한 게이팅·Storage·앱 재시작·데이터 초기화·폰트 번들 등 앱 전반의 비기능 요구사항을 묶은 도메인.

## 배경

데스크탑 앱은 OS 권한·로컬 데이터·폰트 같은 플랫폼 의존 요소들이 많다. 이걸 한 도메인으로 묶어 일관 처리한다.

## 안 하면 어떻게 되는가

- 권한 처리가 모호하면 측정 자체가 동작 X
- 데이터 초기화가 위험하면 사용자 실수로 데이터 소실
- 폰트가 CDN이면 오프라인 환경에서 UI 깨짐

## 사용자와 규모

본인 1인.

## 성공 기준

- ✅ 마이크 + 접근성 권한 둘 다 받기 전엔 메인 진입 차단
- ✅ 앱 재시작 시 진행 중 세션 자동 discard, 잔디·투두·설정은 정상 복원
- ✅ 데이터 초기화는 "모하" 타이핑 friction 거쳐야 동작
- ✅ Pretendard 오프라인 동작 보장
- ✅ 라이트 모드 only
