# timer (뽀모도로 + 세션)

평상시(Idle) ↔ 집중(Focus) ↔ 휴식(Break) ↔ 종료(Complete) 모드 머신과 세션 라이프사이클.

## 배경

뽀모도로 시간 단위로 측정·잔디 적재의 단위가 결정된다. 잘못 끊기면 잔디가 어그러지므로 모드 전환·중단 처리가 정확해야 한다.

## 안 하면 어떻게 되는가

세션 단위가 흐릿하면 잔디 통계가 부정확해지고, 도중 종료/슬립 처리가 일관되지 않으면 사용자 신뢰가 깨진다.

## 사용자와 규모

본인 1인 (개인 프로젝트).

## 성공 기준

- ✅ Idle ↔ Focus ↔ Break ↔ Complete 전이 결정적
- ✅ 도중 취소·재시작·슬립이 모두 자동 discard로 일관 처리
- ✅ 진행 중 OS 알림 미발송, 휴식 시작/완료 시점에만 알림
- ✅ 우상단 chip 텍스트가 모드별로 정확히 표시
