# 모하심 (Mohashim) — v1.0 MVP 기능 명세서

## 1. 제품 개요
* **제품명**: 모하심 (Mohashim)
* **플랫폼**: macOS 및 Windows 데스크탑 앱
* **핵심 가치**: 사용자 조작(키보드/마우스)과 주변 소음(마이크)을 결합한 하이브리드 집중도 측정, 깃허브 스타일의 잔디 통계, 그리고 감자 캐릭터 '모하'의 츤데레/플러팅 멘트로 동기부여를 제공하는 생산성 앱.

---

## 2. 모드별 동작 및 상태 텍스트

### A. 평상시 모드 (Idle Mode)
* **기능**: 투두리스트 관리 및 주변 소음 모니터링 (키보드/마우스 추적 끔).
* **소음 경고**: 80dB 초과 시 소음 경고 멘트 노출.
* **우측 상단 상태 텍스트 (아래 중 무작위 1개 출력, 이모티콘 없음)**:
  * 음료 홀짝이는 중 / 웹 서핑 중 / 멍때리는 중 / 애인 생각 중 / 딴 생각 중 / 상상 중 / 명상 중

### B. 집중 모드 (Pomodoro Mode)
* **세션 기준**: [사용자 설정 집중 시간] + [사용자 설정 휴식 시간] = 1 세션.
* **측정 방식**: 하이브리드 로직 가동 (작업 활동 80점 + 소음 환경 20점).
* **우측 상단 상태 텍스트 (고정, 이모티콘 없음)**:
  * 집중 중

---

## 3. 하이브리드 집중도 산출 로직 (총 100점 만점)
* **작업 활동 점수 (최대 80점)**
  * 글로벌 키보드/마우스 이벤트 감지 (입력 내용 수집 절대 불가, 발생 여부만 체크).
  * **3분 스마트 유예(Grace Period)**: 입력이 멈춰도 인강 시청이나 독서 시간으로 간주하여 3분간은 80점 유지.
  * 3분 초과 시 딴짓/자리 비움으로 간주하여 지속적 감점. (입력 발생 시 즉시 타이머 리셋 및 점수 복구)
* **소음 환경 점수 (최대 20점)**
  * 마이크 데시벨(dB) 측정 및 EMA(지수 이동 평균) 필터링.
  * 0 ~ 65dB: 백색 소음 인정 (20점 만점).
  * 66 ~ 80dB: 데시벨에 비례하여 19점 ~ 1점으로 차등 감점.
  * 80dB 이상: 집중 불가 굉음 (0점).

## 4. 투두리스트 (To-do List) 및 태그 시스템
* **투두리스트 운영**: 뽀모도로 타이머와 완벽히 독립적으로 작동.
* **태그 관리 (설정 연동)**: 
  * '작업 태그' (예: 개발, 과제, 디자인) 및 '위치 태그' (예: 집, 카페, 도서관) 사용자 커스텀 가능.
* **할 일 등록**: 항목 추가 시 작업 태그 1개, 위치 태그 1개를 선택적으로 지정 가능.
* **자동 정렬(Sorting) 로직**:
  * 미완료 항목 (Active): 목록 최상단에 위치.
  * 완료 처리 (Checked): 클릭 즉시 취소선 처리와 함께 목록 **최하단**으로 이동.
  * 롤백 처리 (Unchecked): 완료 항목을 다시 클릭하면, 미완료 상태로 전환되며 목록 **최상단(맨 위)**으로 즉시 복귀.

## 5. 잔디 통계 (Contribution Graph) & 공유
* **잔디 데이터 저장**: 세션이 완전히 끝났을 때(집중+휴식) 해당 세션의 '평균 집중 점수'를 로컬 DB에 저장.
* **잔디 레벨 계산 (최근 28일 기준)**
  * Level 0 (빈칸): 세션 없음
  * Level 1 (연한 색): 세션 1~2개 완료
  * Level 2 (중간 색): 세션 3개 이상 & 평균 60점 미만
  * Level 3 (진한 색): 세션 3~5개 & 평균 60점 이상
  * Level 4 (가장 진한 색): 세션 6개 이상 & 평균 70점 이상
* **내 잔디 자랑하기**: 통계 화면의 공유 버튼 클릭 시, 최근 4주 잔디 그래프와 현재 모하 캐릭터를 포함한 정방형 이미지를 클립보드에 캡처 및 복사.

## 7. 설정 (Settings) 및 데이터 관리
* **타이머 커스텀**: 집중 시간(분) 및 휴식 시간(분) 사용자 직접 설정 가능.
* **태그 커스텀**: 작업/위치 태그 추가 및 삭제 기능.
* **모든 데이터 초기화**: 데스크탑 앱 삭제 시 데이터 잔여 문제를 해결하기 위해, 클릭 시 로컬 DB(잔디, 투두, 설정 등)를 완전히 삭제하는 기능 제공.

---

## 8. 기술 스택
* **Framework**: Tauri v2, React 18, TypeScript, Tailwind CSS
* **Backend Logic**: Rust (rdev 글로벌 이벤트 후킹, 오디오 스트림 제어)
* **Storage**: tauri-plugin-store (오프라인 로컬 JSON 저장)
